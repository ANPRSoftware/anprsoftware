## ANPR.Software

AI-powered license plate recognition for GCC countries.
Supports all GCC countries as UAE, Saudi Arabia, Qatar, Kuwait, Bahrain, and Oman. 
UAE emirate formats, Arabic script OCR, plate number, plate code, country, emirate (in UAE), type (private, commercial, export etc.), plate size, plate color, plate text color.
